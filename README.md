# Assam Tea Company Website

Initial Streamlit Home page preview for Assam Tea Company.

The running quality-message banner appears between the hero area and customer feedback section.

## Run the preview

```bash
python -m pip install -r requirements.txt
streamlit run app.py
```

## Add the selected hero image

Place the selected wide tea garden background image at:

```text
assets/hero-background.jpg
```

The Home page automatically uses that image. Until it is added, a matching green and cream background is displayed.

All visible website text and temporary reviews are editable in `data/business.json`.
